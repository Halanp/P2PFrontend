Project Overview
Build a YouTube clone frontend using React.js with a modern, responsive UI that replicates YouTube's core functionality and user experience.

Technical Stack
React 18+ with TypeScript

State Management: Zustand or Redux Toolkit

Styling: Tailwind CSS with custom YouTube-like theme

Routing: React Router DOM v6

API: Axios + React Query for data fetching

Video Player: React Player or custom Video.js

Icons: React Icons + custom SVG icons

Build Tool: Vite

Testing: Jest + React Testing Library
